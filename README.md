# API Security Risk Analysis

## Overview
This project analyzes the security risks of a public demo API and identifies common API security issues.

## API Tested
JSONPlaceholder Test API

Base URL:
https://jsonplaceholder.typicode.com

## Tools Used
- Postman
- Web Browser

## Methodology
1. Selected a public demo API
2. Tested API endpoints using Postman
3. Inspected response data
4. Identified security risks
5. Classified risk severity
6. Suggested remediation steps

## Endpoints Tested
/posts  
/users  
/comments

## Identified Risks
- Unauthenticated API Access
- Excessive Data Exposure
- Missing Rate Limiting

## Conclusion
The analysis highlights common API security risks that can occur in modern applications and provides recommendations to improve API security.
